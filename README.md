# Proyecto final de POO en la Universidad Nacional de Colombia 
## Problema planteado (Alternativa 1 "Modificada")
Una tienda de videojuegos y perifericos relacionados al gaming necesita de un programa de gestion de inventario debido a la falta de organizacion con la que cuentan y se busca implementar un "sistema de trueques" a este mismo programa para darle mas visibilidad a la tienda
## Solucion al problema planteado
Se plantea una estructura de modulos en la cual se crea el sistema de inventario y se plantea un arhivo en el cual se almacenara (o creara) el producto seleccionado o deseado por el cliente (o gestor) para su organizacion ademas de crear un archivo adicional para la implementacion de el sistema de trueques o intercambios
